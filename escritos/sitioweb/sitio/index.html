1. Cómo hacer un sitio web efectivo

Usualmente se cree que el montaje de un sitio web solamente lo puede hacer un ingeniero de Sistemas, se busca mostrar cómo se puede organizar un equipo de trabajo interdisciplinario para distribuir las tareas y llevar a cabo un trabajo efectivo.

Un sitio web siempre debe tener un propósito, y para buscar el propósito del sitio es indispensable centrar la atención en la información y los servicios que se desean prestar, en este sentido solamente quienes desean montar el sitio, son las que pueden guiar qué tipo de información es relevante colocar o no.

En este mismo sentido se tiene que saber cuáles son los factores más importantes : el tiempo, el costo y la mantenibilidad.

Estas tres variables están intimamente relacionadas y dependiendo de cuál es más importante, se le puede dar a una u otra más importancia.
