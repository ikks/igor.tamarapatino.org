4. Explicando términos técnicos

Dominio

Es el nombre con los cuáles los humanos los identificamos, por ejemplo debian.org, gfc.edu.co, y que identifican de forma única al sitio. Hay empresas dedicadas a garantizar que los nombres son asignados a las organizaciones correspondientes, por ejemplo en Colombia la asignación de nombres la hace la Universidad de los Andes, aunque uno puede comprar .org , .com a otras entidades.

hosting

Es el espacio en una máquina que permanece encendida y conectada a Internet para almacenar los datos que va a tener un sitio web. Además es posible que tenga capacidades para consultar bases de datos e intercambiar servicios con otros sitios.

Sitios estáticos

Son aquellos cuyo contenido está escrito de forma tal que siempre que se visite se va a ver exactamente lo mismo. Por ejemplo el sitio no consulta datos para mostrarlos de acuerdo a un criterio de búsqueda. No tiene nada que ver con que se muevan imágenes o que el mouse cambie de forma.

Sitios dinámicos

Esta clase de sitios hace consultas sobre un sistema de información, un ejemplo clásico es el servicio de consulta de libros de una biblioteca, o un sitio de compras.

Correo electrónico

Servicio que permite establecer comunicación entre usuarios, es usual que cuando se compre un dominio se tenga también cuentas de correo para las personas que estarán manejándolo, por ejemplo, si se registra elparquemasecologicodetodos.org se podrían tener correos electrónicos como :

  • visitas@elparquemasecologicodetodos.org

  • info@elparquemasecologicodetodos.org

  • webmaster@elparquemasecologicodetodos.org

  • pepito@elparquemasecologicodetodos.org
