Desarrollé un proyecto de software libre: GFPP(Flower Production Predictor). Mi intención es mostrar cómo se puede desarrollar software libre. A las empresas realmente no les debería interesar el tipo de licencia del software que usan. Deberían preocuparse claramente por elegir la libertad de escoger a quien quieran para darles soporte y no depender de ninguna otra....(Artículos en este sentido hay bastantes, puede visitar por ejemplo a Erick Raymond o Richard Stallman)
